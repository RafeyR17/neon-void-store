

## ✨ Key Features

- **Cosmic purple gradient** with slow breathing animation
- **Glassmorphic cards** + glowing borders + inset shadows
- **Floating radial orbs** + tsParticles starry background with mouse parallax & connect lines
- **Custom glowing cursor** with trail
- **Scroll-triggered animations** (GSAP + Framer Motion)
- **Fully responsive** (mobile-first, smooth scaling)
- **Perfect accessibility** (contrast, keyboard nav, reduced-motion support)
- **SEO optimized** (Next.js metadata, Open Graph tags)

## 🖥️ Sections

- **Hero** — Oversized glowing name, tagline, hook, photo avatar, CTAs
- **Stats** — Animated counters (50+ projects, global ranking, clients)
- **Skills / Toolkit** — Floating glass orbs with tech icons & tooltips
- **Projects** — Glass cards with live demos, GitHub links, hover depth
- **Journey** — Animated timeline (CS50, tournament win, GitHub Top 50)
- **Testimonials** — Client quotes with metrics
- **Contact** — Glass form + social links

## 🚀 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS + custom glassmorphism
- **Animations**: GSAP (ScrollTrigger) + Framer Motion
- **Particles/Background**: tsParticles + CSS gradient keyframes
- **Icons**: React Icons / Lucide React
- **Deployment**: Vercel (automatic previews & domains)
- **Other**: React Hook Form (contact), Lenis (smooth scroll)
